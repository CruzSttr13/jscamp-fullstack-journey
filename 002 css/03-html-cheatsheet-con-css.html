HTML Cheatsheet

Apunte rápido de etiquetas básicas de HTML.

Estructura básica

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Título</title>
  </head>
  <body>
    Contenido visible
  </body>
</html>

Texto y encabezados

  • <h1> a <h6>: títulos.
  • <p>: párrafo.
  • <br>: salto de línea.
  • <strong>: énfasis fuerte.
  • <em>: énfasis en entonación.

Listas

  • <ul>: lista desordenada.
  • <ol>: lista ordenada.
  • <li>: ítem de lista.
<ol>
  <li>Primer paso</li>
  <li>Segundo paso</li>
</ol>

Enlaces e imágenes

  • <a>: enlace.
  • <img>: imagen.
<a href="https://ejemplo.com">Ir a ejemplo</a>
<img src="imagen.jpg" alt="Descripción">
